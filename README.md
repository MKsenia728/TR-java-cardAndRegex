# Payment Card Validator

A small Java project for generating, reading and validating payment card data using regular expressions.

## Features

- Generate cardholder data using predefined patterns
- Read card data from a file
- Write processed data to a file
- Validate card-related fields using regular expressions
- Unit tests for validation logic

## Tech Stack

- Java
- Regular Expressions
- File I/O
- JUnit

## Purpose

The goal of this project is to practice working with file input/output, regular expressions and validation logic in Java.

## Output Example

The project writes processed card data to `outputCardData.txt`.
